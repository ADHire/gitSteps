Once git init has been ran, you may have noticed there is an invisible folder in the directory now called .git. This folder will track this project's changes. We will talk about that later...

The next step, and one that we constantly will be doing, is to check the status of this local repository. To do this, we will use the command: git status.

When running this command, the terminal will give some output. It will say: Untracked Files: and list some files in a red or green font color. We will touch on the details at a later time. The biggest thing to notice is that all of the files within your directory should be listed here and gives you another hint on what to do next...nothing added to commit but untracked files present (use "git add" to track)

In order for Git to allow us to commit or push these files, we need to add them to tracking. That is our next step!

Commands used so far:

  • git init
  • git status
